# TyporaThemes
 自用的Typora主题

- `essay`: 论文风格主题
> 从知乎上看到的主题, 字体 & 行距都偏大, 适合水论文.

- `latex` 为排版风格主题
> 从github上看到的, 原主题的字体全设成了微软雅黑, 不用另外安装字体.
---
> 雅黑在win10上显示效果比较好, 但是它有版权.
> 
> 商用可以改成思源黑体/宋体.
> 
> 宋体打印效果比黑体好, 尤其是字体偏小的时候, 黑体容易看不清.


- `lazyprint` 背景色都是透明的, 方便打印

- `blackcode` 代码区为黑色, 好看一些

- `drake` Typora官网主题
> 互换了高亮和加粗样式
